# TensrFlow Lite のお試しプログラム類

## ディレクトリ構成

| ディレクトリ    | 内容                                                                 |
|-----------------|----------------------------------------------------------------------|
| download_models | インターネット上で配布されているモデルファイルをダウンロードする     |  
| images          | テスト用画像保存用                                                   |    
| mk_tflite_ssd   | インターネット上で配布されているモデルファイルをtflite形式に変換する |  
| ssd             | SSDを実行する                                                        |
